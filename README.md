# Introduction
Personal vim config files.

* Shortcut Key List:
```
F2 ------ Tag List (Variable/Function/Macro...)
F3 ------ File Explorer
F4 ------ MRU: Most Recent Used File List
F5/F6 --- Gererate tags in current dir
```

# Install
```
git clone git@github.com:iamcopper/vimcfg.git

cd vimcfg

./install.sh
```
